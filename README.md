# Brat
anonymous discussions for college students

## Tech-stack:
 - typescript (language)
 - react-native/expo (frontend)
 - jotai (state)
 - font-awesome (icons)
 - nodejs (backend)
 - mongodb (database)
 - socket.io (glue)
